Be More with Less, Hypergraph Attention Networks for Inductive Text Classification

2020-11-30

概述

这是发表于EMNLP 2020的一篇文章，主要对当前GNN模型存在的问题进行了改进。当前主流的GNN模型通常采用整个语料库中的单词对之间的共现特征作为图上节点的边关系，且在模型训练以前需要在整个语料库层面构建图结构，这带来了两个很严重的问题（文章的动机）：（i）模型无法捕获单词之间更高阶的交互特征；（ii）模型需要很大的计算开销。

针对上述两个问题，文章受最近的超图神经网络[1,2,3]的启发，利用对偶注意力机制，提出了一种归纳文本分类模型HyperGAT，以此捕获单词之间的高阶交互特征，并降低模型计算复杂度。

超图的定义

一个超图记为G=(V,E)G=(\mathcal{V},\mathcal{E})G=(V,E)，其中V={v1,...,vn}\mathcal{V}=\{ v_{1}, ..., v_{n}\}V={v1​,...,vn​}表示图上的节点集合，E={e1,...,em}\mathcal{E}=\{ e_{1}, ..., e_{m}\}E={e1​,...,em​}表示超边集合。需要注意的是，超图上的边能连接两个或者更多节点（σ(e)≥2\sigma(e)\geq 2σ(e)≥2）。

一个超图的典型结构由一个关联矩阵A∈Rn×m\mathbf{A} \in \mathbb{R}^{n \times m}A∈Rn×m表示：

Aij={1, if vi∈ej0, if vi∉ej\mathbf{A}_{i j}=\left\{\begin{array}{l}1, \text { if } v_{i} \in e_{j} \\0, \text { if } v_{i} \notin e_{j}\end{array}\right. Aij​={1, if vi​∈ej​0, if vi​∈/​ej​​

超图中的每个节点都被表示为一个ddd维度的分布向量，因此所有的节点分布可以被表示为X=[x1,...,xn]T∈Rn×d\mathbf{X}=[\mathbf{x}_{1},...,\mathbf{x}_{n}]^{T} \in \mathbb{R}^{n \times d}X=[x1​,...,xn​]T∈Rn×d，因此超图GGG又可以表示为G=(A,X)G=(\mathbf{A}, \mathbf{X})G=(A,X)。

模型对异构特征进行了建模，模型引入多关系超边，主要为序列特征和语义特征，分别对两种特征构建了两个超边。

Sequential Hyperedges

序列化上下文信息能表示单词之间的局部共现性，因此文章引入序列上下文信息来构建超边。传统的序列上下文信息在构建的时候，是通过设定固定大小的滑动窗口来统计单词对之间的共现特征的，但是文章受到[4]的启发，直接把每个句子视作一条超边来连接整个句子中的单词。

Semantic Hyperedges

为了丰富每个单词的语义上下文特征，文章构建了语义超边。

对于语义超边，文章首先使用LDA从文本文档中挖掘潜在的主题TTT，每个主题ti=(θ1,...,θw)t_{i}=(\theta_{1},...,\theta_{w})ti​=(θ1​,...,θw​)（其中www表示词汇表大小）都被表示成整个单词表上的一个概率分布，然后将每个topic作为语义上的超边，以连接文档中有最大概率的top-K个单词。

除了以上两种特征，模型的架构能先天上具有扩展其他特征的特性。

不同于传统的GNN模型，HyperGAT模型为了捕获异构高阶上下文信息，定义了两种不同的聚合方程。

Hypergraph Attention Networks

两种不同的聚合方程：

hil=AGGRedgel(hil−1,{fjl∣∀ej∈Ei})fjl=AGGR⁡nodel({hkl−1∣∀vk∈ej})\begin{aligned}\mathbf{h}_{i}^{l} &=\mathrm{AGGR}_{e d g e}^{l}\left(\mathbf{h}_{i}^{l-1},\left\{\mathbf{f}_{j}^{l} \mid \forall e_{j} \in \mathcal{E}_{i}\right\}\right) \\\mathbf{f}_{j}^{l} &=\operatorname{AGGR}_{n o d e}^{l}\left(\left\{\mathbf{h}_{k}^{l-1} \mid \forall v_{k} \in e_{j}\right\}\right)\end{aligned} hil​fjl​​=AGGRedgel​(hil−1​,{fjl​∣∀ej​∈Ei​})=AGGRnodel​({hkl−1​∣∀vk​∈ej​})​

其中，AGGRedge\mathrm{AGGR}_{edge}AGGRedge​函数将超边上的特征聚合到节点上，AGGRnode\mathrm{AGGR}_{node}AGGRnode​函数将节点上的特征聚合到超边上。
