Topic Memory Networks for Short Text Classification

概述

短文本分类因为数据稀疏特性，使得很多模型在该领域的性能表现不是很好（动机）。以下Table 1为例：

从Table 1可以看到，在没有充分的上下文的情况下，模型倾向于将测试样例划分到R1，而不是R2，即模型倾向于把句子划分给句子结构相似的类别，这样子的误分类样本广泛存在于诸如CNN、LSTM之类的深度网络模型。

为了解决这一问题，有两种处理方法：知识增强和主题模型方法（文章的方法）。知识增强方法的核心是引入外部知识以解决数据稀疏问题，但是这会有不同语义空间的向量融合的新问题，且不同语言或者领域无法应用；主题模型的核心是用推理的方式来挖掘数据中更深层次的表示，这方面比较经典的应用就是VAE。

为了解决这一问题，文章提出了一种主题记忆网络（TMN）用于短文本分类，该模型基于主题记忆机制来编码潜在的主题表示，以表示类别标签。不同于之前研究着眼于引入外部知识库的方式扩展特征，该模型则是采用端到端的方式，联合探索主题索引和文本分类。

Model

模型概述

TMN模型的整体架构如Figure 1所示：

主要分成三个部分，从左到右依次为：Neural Topic Model、Topic Memory Mechanism和Classifier。

给定数据集X={x1,x2,…,xM}\mathbf{X}=\left\{\mathbf{x}^{1}, \mathbf{x}^{2}, \ldots, \mathbf{x}^{M}\right\}X={x1,x2,…,xM}，共包含MMM个短文本样本，每个样本xxx都被表示成两种表示：词袋term向量xBoW∈RV\mathbf{x}_{B o W} \in \mathbb{R}^{V}xBoW​∈RV和单词索引序列向量xSeq∈RL\mathbf{x}_{S e q} \in \mathbb{R}^{L}xSeq​∈RL，其中VVV为词汇表大小，LLL是文本序列长度。xBoW\mathbf{x}_{B o W}xBoW​作为Neural Topic Model的输入用于诱导产生主题，然后与xSeq\mathbf{x}_{S e q}xSeq​在Topic Memory Mechanism部分相匹配，以学习分类特征。

Neural Topic Model(NTM)

NTM部分基于VAE模型（这个可以读一下VAE的原始论文，不在这里展开，只简单说一下原理），包含一个连续潜在变量z∈RKz \in \mathbb{R}^{K}z∈RK作为一种中间表示，其中KKK表示主题数目。NTM和VAE一样，模型分成两个部分：编码模块和解码模块（生成部分），神经网络通过对数据的学习，得到一个中间表示zzz，然后再通过解码的方式，得到与原始数据近似的数据。

NTM Generation

模型假设每个样本x\mathbf{x}x都包含一个KKK维度的主题混合度θ\thetaθ，该混合度由高斯分布产生，每个主题kkk都由词汇表中的单词分布ϕk\phi_{k}ϕk​表示。对样本x\mathbf{x}x的生成过程有如下几步：

  • 生成连续潜在变量：z∼N(μ,σ2)\mathbf{z} \sim \mathcal{N}\left(\boldsymbol{\mu}, \boldsymbol{\sigma}^{2}\right)z∼N(μ,σ2)
  • θ=softmax⁡(fθ(z))\theta=\operatorname{softmax}\left(f_{\theta}(\mathbf{z})\right)θ=softmax(fθ​(z))
  • 对于x\mathbf{x}x中的第nnn个单词，得到单词分布 Draw word wn∼softmax⁡(fϕ(θ))\text { Draw word } w_{n} \sim \operatorname{softmax}\left(f_{\phi}(\theta)\right) Draw word wn​∼softmax(fϕ​(θ))。

其中，f∗(⋅)f_{*}(\cdot)f∗​(⋅)是神经感知器，由一层全连接和激活函数构成，这里采用Relu作为激活函数，先验参数μ\boldsymbol{\mu}μ，σ\boldsymbol{\sigma}σ，zzz都由输入数据估计得到，采用下面的估计方程：

μ=fμ(fe(xBoW)),log⁡σ=fσ(fe(xBoW))\boldsymbol{\mu}=f_{\mu}\left(f_{e}\left(\mathbf{x}_{B o W}\right)\right), \log \boldsymbol{\sigma}=f_{\sigma}\left(f_{e}\left(\mathbf{x}_{B o W}\right)\right) μ=fμ​(fe​(xBoW​)),logσ=fσ​(fe​(xBoW​))

与VAE不同的地方也在这里，文章引入了参数θ\thetaθ和ϕ\phiϕ增强了模型的可解释性。

NTM Inference

模型采用变分推理，用差分下界的负数作为损失函数，即：

LNTM=DKL(q(z)∥p(z∣x))−Eq(z)[p(x∣z)]\mathcal{L}_{N T M}=D_{K L}(q(\mathbf{z}) \| p(\mathbf{z} \mid \mathbf{x}))-\mathbb{E}_{q(\mathbf{z})}[p(\mathbf{x} \mid \mathbf{z})] LNTM​=DKL​(q(z)∥p(z∣x))−Eq(z)​[p(x∣z)]

这部分的推理可以看看我读VAE的论文笔记。

Topic Memory Mechanism

主题记忆机制用于将NTM产生的潜在主题映射到分类特征，它有两个来源输入，一个来自NTM部分，一个是xSeq\mathbf{x}_{S e q}xSeq​。为了实现这个功能，TMM设计了源记忆矩阵SSS和目标记忆矩阵TTT，都是K×EK \times EK×E大小的，其中KKK是主题数目，EEE是词向量维度。SSS和TTT通过两个ReLU激活的神经感知器产生，且都以主题-单词权重矩阵Wϕ∈RK×V\boldsymbol{W}^{\phi} \in \mathbb{R}^{K \times V}Wϕ∈RK×V作为输入，是fϕ(⋅)f_{\phi}(\cdot)fϕ​(⋅)的参数矩阵。

这一机制分成两个阶段，假设UUU表示xseq\mathbf{x}_{seq}xseq​的嵌入表示：（1）源记忆阶段，模型通过下式计算第kkk个主题和xseq\mathbf{x}_{seq}xseq​中第lll个单词嵌入之间的匹配：

Pk,l=sigmoid⁡(Ws[Sk;Ul]+bs)\boldsymbol{P}_{k, l}=\operatorname{sigmoid}\left(\mathbf{W}^{s}\left[\boldsymbol{S}_{k} ; \boldsymbol{U}_{l}\right]+b^{s}\right) Pk,l​=sigmoid(Ws[Sk​;Ul​]+bs)

其中，[x;y][\mathbf{x} ; \mathbf{y}][x;y]表示x\mathbf{x}x和y\mathbf{y}y的concate操作，其余为参数。为了进一步与PPP结合样本主题混合度θ\thetaθ，文章定义了整合记忆权重：

ξk=θk+γ∑lPk,l\xi_{k}=\theta_{k}+\gamma \sum_{l} \boldsymbol{P}_{k, l} ξk​=θk​+γl∑​Pk,l​

其中，γ\gammaγ是预定义的相关系数。然后通过权重目标矩阵TTT结合ξ\xiξ得到TMM的输出表示RRR。

最后将RRR和UUU的concate作为classifier的输入特征。个人认为TMM机制和Attention机制非常的相似，ξ\xiξ的本质就是attention机制中的α\alphaα，只是与Attention机制不同的是，ξ\xiξ加上了其他的参数，这应该是TMN能捕获样本中，与分类有关的指示性单词的原因。

Joint Learning

因为模型分成两个部分，即NTM部分和分类器部分，文章借鉴了联合学习的思想，构建了整个模型的联合损失：

L=LNTM+λLCLS\mathcal{L}=\mathcal{L}_{N T M}+\lambda \mathcal{L}_{C L S} L=LNTM​+λLCLS​

式(5)右边第一项表示NTM部分的损失，第二项表示分类器的交叉熵损失，λ\lambdaλ是用于控制主题模型和分类之间的平衡参数。
